# 环境配置

## 1.配置申请智能体身份证书的环境

CA客户端目前支持MacOS和Linux操作系统，您需要确保系统中安装了以下工具：

- `openssl` - 用于密钥和证书操作
- `curl` - 用于 HTTP 请求
- `jq` - 用于 JSON 处理
- `base64` - 用于 Base64 编码 (通常系统自带)

### 安装依赖 (示例)

**macOS:**

```bash
brew install openssl curl jq
```

**Ubuntu/Debian:**

```bash
sudo apt-get update
sudo apt-get install openssl curl jq
```

**CentOS/RHEL:**

```bash
sudo yum install openssl curl jq
```
